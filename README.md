# 🌸 Flower Classification using Deep Learning

This project demonstrates flower image classification using:

- Custom CNN (from scratch)
- MobileNetV2 (Transfer Learning)

## Supported Classes
- Daisy
- Dandelion
- Roses
- Sunflowers
- Tulips

## Features
- Real-time image upload
- Model comparison
- Confidence-based final prediction
- Streamlit UI
- Production-style deployment

## Tech Stack
- Python
- TensorFlow / Keras
- Streamlit
- Google Drive (model hosting)
